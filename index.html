The Gem Game

  • You will be given a random number at the start of each game.
  • There are four crystals below. Clicking on each crystal will add a specific amount of points to your total.
  • You win the game by matching the random number you are given at the start of the game. You lose the game if you go over that number.
  • The value of each crystal is hidden until you click on it.
  • Each time the game starts, the value of each crystal will change.
  • Don't be a LOSER!
